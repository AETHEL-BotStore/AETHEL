Пользовательское соглашение для клиентов

Настоящее Соглашение определяет условия использования сервиса по записи на услуги мастеров индустрии красоты.

1. Термины и определения

Сервис — программное обеспечение (telegram-бот), предоставляющее возможность записи на услуги мастеров.

Пользователь — физическое лицо, использующее Сервис для записи на услуги.

Мастер — лицо, оказывающее услуги, с которым Пользователь заключает договор на оказание услуг.

Администратор Сервиса — ИП Прокудин Матвей Евгеньевич, обеспечивающее техническую работу Сервиса.

2. Общие положения

2.1. Используя Сервис, Пользователь подтверждает, что ознакомлен и согласен с условиями настоящего Соглашения.

2.2. Администратор Сервиса предоставляет техническую возможность для записи, но не является стороной договора между Пользователем и Мастером.

2.3. Пользователь подтверждает, что ему исполнилось 18 лет, либо он имеет согласие законных представителей.

3. Регистрация и персональные данные

3.1. Для использования Сервиса Пользователь предоставляет:

  • Имя (псевдоним)
  • Номер телефона

3.2. Пользователь гарантирует достоверность предоставленной информации.

3.3. Персональные данные обрабатываются в соответствии с Политикой конфиденциальности для клиентов.

4. Запись и оплата

4.1. Запись на услугу осуществляется через интерфейс Сервиса.

4.2. Условия предоплаты (размер, порядок возврата) устанавливаются Мастером и отображаются при записи.

4.3. При оплате через платежную систему Т‑Банка применяются правила соответствующей платежной системы.

4.4. Администратор Сервиса не несет ответственности за качество оказываемых Мастером услуг.

5. Возврат предоплаты

5.1. Пользователь имеет право отменить запись через интерфейс Сервиса.

5.2. Условия возврата предоплаты:

  • При отмене за установленное Мастером количество часов до записи — предоплата возвращается
  • При отмене позже установленного срока — предоплата не возвращается
  • При отмене записи Мастером — предоплата возвращается в полном объеме
  • При неявке на запись без предупреждения — предоплата не возвращается

5.3. Возврат осуществляется тем же способом, которым была произведена оплата.

6. Правила поведения

6.1. Пользователь обязуется:

  • Своевременно предупреждать об отмене или переносе записи
  • Не использовать Сервис для спама и мошенничества
  • Не распространять оскорбительную информацию

6.2. За нарушение правил Мастер или Администратор вправе заблокировать доступ Пользователя к Сервису.

7. Заключительные положения

7.1. Администратор Сервиса вправе вносить изменения в настоящее Соглашение с уведомлением Пользователей через Сервис.

7.2. По всем вопросам Пользователь может обращаться в поддержку: @AETHEL_support
