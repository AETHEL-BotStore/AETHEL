1

Выберите подходящую версию бота

Советуем на пробный период выбрать версию Maximum (максимум), чтобы попробовать все функции. В любой момент версию можно изменить в личном кабинете.

Basic

Базовая
Для старта и самых необходимых функций
  • ✅ Кнопка «Список услуг»
  • ✅ Кнопка «Как дойти»
  • ✅ Обращение к мастеру
  • ✅ Часовой пояс
  • ✅ Скрытие дней
  • ✅ Максимальный срок записи
  • ✅ Настройка отмены/переноса

Pro

Продвинутая
Всё из Basic + дополнительные возможности
  • ✅ Всё из Basic
  • ✅ Кнопка «Отзывы»
  • ✅ Кнопка «Важно»
  • ✅ Рассылки клиентам
  • ✅ Общий перерыв
  • ✅ Авто-подтверждение записей
  • ✅ Редактирование напоминаний
  • ✅ Предоплата по ссылке

Maximum

Максимальная
Все функции для полной автоматизации
  • ✅ Всё из Basic + Pro
  • ✅ Блокировка времени мастера
  • ✅ Статистика
  • ✅ Индивидуальные перерывы
  • ✅ Бан пользователей
  • ✅ Опция «Требуется снятие»
  • ✅ Уведомление после записи
  • ✅ Мультизапись
2

Выберите категорию под ваш бизнес

👤 Личный

Бот для одного мастера, который автоматически записывает клиентов, отвечает на частые вопросы и управляет расписанием.

  • ✅ Для одного мастера
  • ✅ Экономит время
  • ✅ Заменяет администратора

🏢 Студия

Система записи для нескольких мастеров. Каждый специалист ведёт своё расписание.

  • ✅ До 5 мастеров
  • ✅ Автоматическое распределение
  • ✅ Исключение накладок

💼 Бизнес

Система управления студией для владельца. Контроль записей, расписаний и финансов.

  • ✅ Неограниченно мастеров
  • ✅ Контроль загрузки
  • ✅ Финансовые показатели

💰 Бизнес + Платежи

С автоматическим приёмом оплат через банковский эквайринг.

  • ✅ Платежи через эквайринг
  • ✅ Авто-подтверждение записи
  • ✅ Чеки по требованию онлайн-кассы

Перейдите в бот-магазин @AETHEL_Store_bot

📌 Нажмите “Выбрать бота” → выберите нужную категорию → выберите версию → нажмите “Подключить”

📱 Интерфейс бота-магазина AETHEL_Store_bot
4

Создайте бота в @BotFather

Найдите в Telegram официального бота @BotFather и выполните следующие шаги:

  1. Отправьте команду /newbot
  2. Придумайте имя бота (например, Lash Studio Анна) — его будут видеть пользователи
  3. Придумайте юзернейм бота (например, LashStudioAnnaBot) — должен оканчиваться на ...bot или ..._bot
  4. Сохраните полученный API-токен (длинная строка из цифр и букв) — это ваш ключ доступа
Пример токена: 7234567890:AAGk7xE3fGmH9jKlM2nPqR4sTuV5wXyZ1bC

⚠️ Важно! Никому не передавайте свой API-токен. Это ключ доступа к вашему боту.

5

Введите токен в боте-магазине

Вернитесь в бот @AETHEL_Store_bot и:

  • Вставьте полученный от @BotFather токен
  • Придумайте имя бота (его будете видеть только вы в админ-панели)

✅ Готово! Ваш бот успешно создан и активирован!

6

Получите ссылку на своего бота

Чтобы зайти в вашего бота:

  • Введите в поиске Telegram юзернейм, который вы указали в @BotFather
  • Например: @LashStudioAnnaBot
  • Запустите бота командой /start

📤 Этой ссылкой делитесь с клиентами! Они будут записываться к вам через этого бота.

7

Вход в админ-панель

Напишите своему боту команду:

/admin

Теперь вы можете настраивать услуги, расписание и управлять ботом.
