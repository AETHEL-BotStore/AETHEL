📚 Гайд по использованию админ-панели

Полное руководство по настройке и управлению Telegram ботом для мастеров

@AETHEL_support • 25 декабря 2025

🎯 AETHEL: Полное руководство по началу работы

Это руководство разделено на две части:

  • Для Покупателя (Главного Администратора): Вы приобрели бота и настраиваете его с нуля.
  • Для Мастера: Вас добавили в уже работающего бота.

ЧАСТЬ 1: ДЛЯ ПОКУПАТЕЛЯ (ГЛАВНОГО АДМИНИСТРАТОРА)

Вы купили бота и вам нужно его создать и настроить.

🛠 Шаг 1: Создание бота в @BotFather

Найдите в Telegram официального бота @BotFather.

  1. Отправьте ему команду /newbot.
  2. Следуйте инструкциям:
    • Придумайте имя бота (например, Lash Studio Анна). Это имя, которое будут видеть пользователи.
    • Придумайте юзернейм бота (например, LashStudioAnnaBot). Он должен оканчиваться на ...bot или ..._bot.
  3. После создания @BotFather выдаст вам API-токен — длинная строка из цифр и букв. Сохраните его!

⚠️ Важно! Никому не передавайте API-токен — это ключ доступа к вашему боту.

⚙️ Шаг 2: Настройка бота в магазине AETHEL

Перейдите в бота-магазин, где вы приобрели тариф (AETHEL | Личный, AETHEL | Салон или AETHEL | Бизнес).

Введите полученный от @BotFather токен и имя бота по запросу.

💡 После успешной привязки токена ваш бот активируется. Чтобы перейти в своего бота, просто введите в поиске Telegram его юзернейм (например, @LashStudioAnnaBot) и запустите командой /start.

🔑 Ваши права после покупки

Вы автоматически становитесь Главным Администратором.

  • AETHEL | Личный: Вы — единственный мастер. Добавление других мастеров недоступно.
  • AETHEL | Салон: Вы — Главный Администратор и мастер. Вы можете добавлять других мастеров в систему.
  • AETHEL | Бизнес: Вы — Главный Администратор с полным правом управления всеми мастерами, их расписанием и услугами.

Теперь вы можете настроить контент бота и добавить мастеров, если ваш тариф это позволяет.

ЧАСТЬ 2: ДЛЯ МАСТЕРА (КОТОРОГО ДОБАВИЛИ)

Вас добавили в готового бота. Ваша цель — настроить свой профиль и рабочее расписание.

🚀 1. Получение доступа к админ-панели

Главный администратор добавляет вас в систему командой в боте:

/admin → "Управление мастерами" → "➕ Добавить мастера"

Для этого потребуется ваш Telegram ID (можно получить, написав команду /myid в вашем боте).

После добавления вы получите доступ к админ-панели.

2. Вход в админ-панель

Напишите боту команду /admin. Откроется главное меню с доступными функциями.

🎛 ОБЗОР АДМИН-ПАНЕЛИ

🔧 Админ-панель ч.1 (основное меню)

  • Далее➡️ - переход к расширенным функциям
  • Мой профиль - настройка вашей информации
  • Бан пользователей - блокировка клиентов
  • Отправить рассылку - массовые уведомления
  • Настроить бота (только для главного админа)
  • Управление мастерами (только для главного админа)
  • 📖 Мануал - это руководство

🔧 Админ-панель ч.2 (управление расписанием)

  • ⬅️ Назад - возврат в главное меню
  • Посмотреть записи - просмотр бронирований
  • Посмотреть расписание - обзор графика
  • Управление расписанием - настройка дней
  • Управление услугами - добавление/редактирование
  • Блокировка времени - ручное закрытие слотов
  • Статистика - аналитика записей и выручки
  • Установить реквизиты - платежные данные

👤 Настройка профиля мастера (Команда "Мой профиль")

Редактирование информации, которую видят клиенты:

  • ✏️ Изменить имя — ваше имя для клиентов
  • 📝 Изменить описание — информация о вас, опыте и специализации
  • 📍 Изменить адрес — место приема клиентов (если выездная, укажите это)
  • 📸 Изменить фото — ваше профессиональное фото

💡 Советы: Все изменения сразу видны клиентам. Используйте качественное фото. В описании укажите опыт, ключевые услуги и преимущества.

📅 Управление расписанием

Создание расписания:

  • Перейдите в "Управление расписанием" → "Составить расписание"
  • Выберите месяц (зеленая галочка ✅ означает, что расписание на этот месяц уже есть)
  • Выберите рабочие дни: 🔵 — есть расписание, ❌ — есть записи (нельзя изменить)
  • Установите время работы и подтвердите выбор

Настройка перерывов:

  • Постоянный перерыв для всех дней
  • Индивидуальные перерывы для конкретных дней
  • Минимальная длительность перерыва — 15 минут

⚠️ Важно: Изменять время работы можно только для дней, на которые еще нет записей.

💼 Управление услугами

Доступные действия:

  • Добавить новую услугу
  • Редактировать существующую (название, описание, цена)
  • Установить размер предоплаты (рекомендуется 30-50% от цены)

Обязательные параметры для каждой услуги:

  • Название — понятное и краткое
  • Описание — детали и особенности
  • Цена — полная стоимость
  • Предоплата — сумма бронирования
  • Длительность — время на выполнение

👥 Работа с записями клиентов

Просмотр записей: "Посмотреть записи" → выберите день → список клиентов с информацией о времени, услуге и стоимости.

Отмена записей: Выберите день и запись, подтвердите отмену. Система автоматически сообщит клиенту.

Статистика: В разделе "Статистика" можно посмотреть количество пользователей, суммарную выручку, выполненные и отмененные записи.

⏰ Блокировка времени

Ручная блокировка слотов: "Блокировка времени" → выберите день и время начала/окончания.

Когда использовать: личные встречи, внеплановые перерывы, технические работы.

💳 Настройка реквизитов

Перейдите в "Установить реквизиты" и введите данные в формате:

  • Номер карты или телефона
  • Название банка (например, Тинькофф, Сбербанк)

⚠️ Эти реквизиты увидят клиенты при оплате предоплаты. Своевременно обновляйте данные при смене карты.

🔔 Рассылка уведомлений

"Отправить рассылку" → напишите текст → при необходимости добавьте кнопку с ссылкой.

Как добавить кнопку: после текста отступите строку и введите название кнопки, затем дефис и ссылку.

💡 Рекомендации: Лучшее время для рассылки — с 11:00 до 16:00 по будням. Не злоупотребляйте, отправляйте только важную информацию.

🚫 Бан пользователей

"Бан пользователей" → выберите пользователя → подтвердите блокировку.

Основания для бана: неадекватное поведение, многократные отмены в последний момент, нарушение правил.

⚙️ Настройка контента бота (ТОЛЬКО для главного администратора)

Доступные разделы для редактирования:

  • Приветствие — сообщение при команде /start
  • Информация — раздел "Важно"
  • Список услуг — общее описание услуг салона
  • Локация — адрес и карта проезда
  • Отзывы — ссылка на канал с отзывами

💡 Советы по оформлению: Используйте emoji для визуального разделения, добавляйте качественные фото работ, регулярно проверяйте актуальность информации и цен.

🆘 Решение проблем

❓ Клиент не может записаться на нужное время?

Проверьте, не заблокирован ли слот, нет ли перерыва, помещается ли время выполнения услуги.

❓ Клиент не может оплатить предоплату?

Проверьте, правильно ли указаны реквизиты в разделе "Установить реквизиты".

❓ Бот не отвечает?

Попробуйте перезагрузить сессию, отправив команды /start и /admin. Если проблема не исчезла, обратитесь в поддержку.
